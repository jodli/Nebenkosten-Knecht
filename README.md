# Der Nebenkosten-Knecht 🚀 - Utility Bill Calculator

Welcome to `Der Nebenkosten-Knecht`! This project is a personal journey to build a custom web application for managing and calculating annual utility bills (the infamous German "Nebenkostenabrechnung") for a specific multi-family home. 🏠💸 It aims to replace a cumbersome spreadsheet process with something more streamlined and tailored.

**Disclaimer:** This is a **highly personalized tool** built for my specific needs and the unique calculation rules of my building. It's not intended as a general-purpose solution (yet 😉).

## What it Does (or will do!) ✨

This app is being built to handle the following:

*   📊 **Track Meter Readings:** Easily log monthly readings for various meters (water, heating components, etc.).
*   💰 **Manage Costs:** Record annual costs associated with the building (heating fuel, maintenance, insurance, taxes, waste disposal, etc.).
*   🧾 **Handle Prepayments:** Keep track of advance payments made by tenants.
*   🧠 **Calculate Automatically:** Crunch the numbers based on predefined, specific rules (e.g., distribution by living area, individual consumption, fixed shares, special conditions).
*   📄 **Generate Reports:** Create a clear, simple overview for each household, showing their share of costs, prepayments made, and the final balance (amount due or credit).

## Tech Stack 🛠️

*   **Backend:** Rust (Actix Web)
*   **Frontend:** React
*   **Data Storage:** Simple file-based storage (JSON)
*   **Configuration:** External config file for building-specific parameters
