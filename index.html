Project Overview

Nebenkosten-Knecht is a utility for landlords to track and calculate utility costs and generate accurate annual utility cost statements (Nebenkostenabrechnungen) for tenants.

Available Features

  • Property Unit Management: Create, edit, and delete property units with living area information
  • Tenant Management: Manage tenants and assign them to property units
  • Meter Management: Define utility meters (electricity, water, etc.) and assign them to units or mark as common

Running the Application

You can start the application using the following commands:

./run.sh

Or run the backend and frontend separately:

./backend/run.sh

and

./frontend/run.sh
